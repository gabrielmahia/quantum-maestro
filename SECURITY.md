# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 12.x    | :white_check_mark: |
| 11.x    | :x:                |

## Reporting a Vulnerability
If you discover a mathematical error in the Risk Calculation or Position Sizing logic:
* **DO NOT** open a public issue.
* Email me directly at: contact@aikungfu.dev

This ensures the integrity of the risk models before a patch is released.
