# Contributing to Quantum Maestro

Thank you for your interest! Currently, this is a **Closed Source / Personal Project**.

## Reporting Bugs
If you find a logic error in the Risk Engine (e.g., incorrect VIX scaling), please open an Issue using the "Bug Report" template.

## Feature Requests
I am open to ideas! Use the "Feature Request" template to suggest new institutional filters (e.g., Dark Pool data integration).

## Pull Requests
Unsolicited PRs will not be merged unless discussed in an Issue first.
